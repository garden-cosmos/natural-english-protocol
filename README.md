# natural-english-protocol
这个skill的角色是中文语境理解 + 自然英文重建，服务的是中文母语者在真实生活里想用英文表达自己时遇到的那些场景：想说得自然一点、礼貌一点、亲切一点、别太硬、别太像翻译软件、别太像 AI。先理解中文背后的真实意图，再将其改写成英语环境里自然、清楚、得体的话。

Skill里面内置了英国政府的《简明英语教程》以及美国政府的《18F科技网站编写指南》，可以把想要表达的中文用地道且不生硬的英文表达出来，改善硅基+机翻的毛病。任何想写的面向用户的英文都可以用这个skill 来输出。

举例：
"此功能帮您省心管理订单"
常规机翻：“This feature helps you save worry managing orders”
写作助手：Set it and forget it — your orders, on autopilot

中文表达：辛苦你了
直译问题：You worked hard 很怪
自然英文方向：Thanks for taking care of this. / Really appreciate your help.

中文表达：麻烦你了
直译问题：I trouble you 错误
自然英文方向：Thanks so much. / I appreciate it. / Would you mind...?

中文表达：不好意思打扰你
直译问题：Sorry to disturb you 太重
自然英文方向：Sorry to bother you. / Hope it’s okay to ask.

中文表达：有空的话帮我看一下
直译问题：直译很绕
自然英文方向：No rush, but could you take a look when you have a chance?

中文表达：我怕说得不清楚
直译问题：直译生硬
自然英文方向：I’m not sure if I explained that well.

中文表达：你方便吗
直译问题：Are you convenient? 错误
自然英文方向：Does that work for you? / Are you free then?

场景写个大概就好，邮件、回复、邮件、产品、帮助文档，什么场景都接得住。
